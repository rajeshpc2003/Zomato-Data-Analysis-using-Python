    <h1>🍽️ Zomato Data Analysis using Python</h1>
    <p class="lead">Data analysis project exploring Zomato restaurant data using <strong>Python</strong> (Pandas, NumPy, Matplotlib, Seaborn). This README is a web/HTML version suitable for hosting or embedding in project pages.</p>

    <h2>📊 Project Overview</h2>
    <p>This project analyzes Zomato's dataset to answer questions such as:</p>
    <ul>
      <li>Which countries have the highest number of restaurants listed?</li>
      <li>What are the most popular cuisines and price ranges?</li>
      <li>How do ratings vary across countries or cuisines?</li>
      <li>What factors influence restaurant ratings?</li>
    </ul>

    <p>Inspired by the GeeksforGeeks tutorial: <a href="https://www.geeksforgeeks.org/data-science/zomato-data-analysis-using-python/" target="_blank" rel="noopener">Zomato Data Analysis using Python</a>.</p>
